# Personal Portfolio Website

This is my personal portfolio website, built to showcase my skills, projects, and experience as a **Front-End Developer**.

The project focuses on delivering a modern, responsive, and visually appealing user experience, highlighting my work and technical abilities.

[Test my site](https://gabrielgamis.github.io/Site-Portfolio-En/)

---

## 🔹 Features

- Responsive design for mobile and desktop
- Interactive navigation menu (desktop & mobile)
- Sections for:
  - Home
  - Skills / Specialties
  - About Me
  - Projects
  - Contact
- Contact form with user input fields
- Social media integration
- Smooth and clean UI design

---

## 🔹 Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Google Fonts (Poppins)  
- Bootstrap Icons
